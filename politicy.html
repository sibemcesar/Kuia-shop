1. Uso da Loja

O que podes fazer na nossa loja

Ao entrares na Kuia+ Shop, tens acesso a várias coisas fixes para tornar as tuas compras mais fáceis e rápidas:

  • Ver tudo à vontade — Podes explorar todos os produtos, ver fotos, descrições e preços sem precisar de criar conta. É só abrir e navegar!
  • Comprar o que quiseres — Encontraste algo que gostas? É só seguir os passos para encomendar e pagar. Simples assim.
  • Organizar do teu jeito — Podes filtrar os produtos por categoria, para encontrares mais rápido aquilo que procuras.
  • Criar a tua conta — Com uma conta, podes finalizar compras, ver o histórico dos teus pedidos e receber alertas de ofertas.
Ao usares a loja, estás a concordar com estas regras. Se não concordas com alguma coisa, é melhor não usar a plataforma.

2. Favoritos e Filtros

As tuas escolhas ficam guardadas no teu telemóvel

Podes marcar produtos como favoritos e usar filtros para encontrar coisas mais rápido. Aqui está o que precisas de saber:

  • Fica tudo no teu aparelho: Os teus favoritos e filtros são guardados só no teu telemóvel ou computador. Ninguém mais tem acesso.
  • Ninguém vê as tuas escolhas: Essas informações não são enviadas para lado nenhum. Não vendemos nem partilhamos nada com ninguém.
  • Tu controlas tudo: Se quiseres apagar os teus favoritos, é só limpar os dados do navegador. Não precisas de pedir a ninguém.
Os teus favoritos são 100% privados. Só o teu aparelho sabe o que tu guardaste — mais ninguém!

3. Privacidade e Dados

Como protegemos as tuas informações

Respeitamos a tua privacidade e tratamos os teus dados com muito cuidado. Aqui está o resumo:

  • Não pedimos coisas sensíveis: Nunca vamos pedir o teu BI, dados bancários ou localização sem te avisar primeiro e sem a tua autorização.
  • Vemos o que é popular: Podemos ver quais categorias e produtos são mais visitados (de forma anónima), só para melhorar a loja e mostrar coisas que te possam interessar.
  • Conta simples: Para criares conta, só pedimos o básico — nome, contacto e foto de perfil. Nada mais.
  • Não vendemos os teus dados: Em nenhuma situação os teus dados são vendidos ou dados a outras empresas. Ponto final.
A tua privacidade é super importante para nós. Só usamos as tuas informações para fazer a loja funcionar melhor para ti.

4. Responsabilidade sobre Produtos

Quem é responsável pelo quê

A Kuia+ Shop é tipo um centro comercial online — nós juntamos os compradores e os vendedores num só lugar. Mas é importante saberes quem faz o quê:

  • Os produtos são dos vendedores: Cada vendedor é responsável pelo que vende — a qualidade, o preço, a descrição e a entrega. Se tiveres algum problema, o vendedor é quem deve resolver.
  • Nós somos o meio de campo: A Kuia+ Shop oferece o espaço para as vendas acontecerem. Fazemos o possível para verificar os vendedores, mas não controlamos cada produto individualmente.
  • Se houver problema, ajudamos: Se algo correr mal com um pedido, podemos ajudar a conversar entre ti e o vendedor para encontrar uma solução justa.
  • Garantias: As garantias dos produtos são dadas pelo vendedor. Antes de comprares, verifica sempre as condições de garantia e devolução.
Dica: antes de comprares, olha sempre a avaliação do vendedor e lê bem a descrição do produto. Assim evitas surpresas!

6. Encomendas e Pedidos

Como funcionam os pedidos e a segurança

Queremos que cada encomenda corra bem, por isso temos algumas regras para garantir que tudo é feito de forma segura e transparente:

  • Recusa de pedidos: A Kuia+ Shop reserva-se o direito de recusar qualquer pedido caso alguma informação não esteja correcta, incompleta, ou pareça suspeita. Isso inclui dados de contacto errados, endereços inválidos ou qualquer inconsistência no pedido.
  • Verificação de identidade: Por questões de segurança, podemos pedir-te uma verificação de identidade antes de processar o teu pedido. Isto protege-te a ti e a nós contra fraudes e garante que a encomenda vai para a pessoa certa.
  • Localização para entrega em casa: Se quiseres receber o produto em casa, precisamos da tua localização actual (morada completa ou localização GPS). Sem esta informação, não é possível fazer a entrega ao domicílio.
  • Confirmação do pedido: Depois de fazeres o pedido, vais receber uma confirmação com os detalhes. Verifica tudo antes de confirmar para evitares problemas.
  • Prazo de entrega: O prazo varia conforme a tua localização e a disponibilidade do produto. Faremos o possível para entregar o mais rápido possível.
IMPORTANTE: Todos os dados de segurança, verificação de identidade e informações do pedido são sempre tratados via número telefónico ou pelo nosso WhatsApp oficial. Nunca partilhes essas informações no Chat da Loja. O chat da loja é apenas para consultas gerais — os pedidos reais são finalizados por telefone ou WhatsApp para tua segurança.
Se alguma informação do teu pedido estiver incorrecta, o pedido será recusado automaticamente até que os dados sejam corrigidos. Isto é para a tua própria protecção!
Lembra-te: os pedidos e verificações são feitos apenas por telefone ou WhatsApp. Nunca no chat da loja! Isso garante mais segurança para ti.
